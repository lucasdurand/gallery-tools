# gallery-tools
Some tools for setting up a multi-user gallery of ... 

- analytics viz?
- bottle cap collections?
- mismatched shoes?

**A gallery should have:**

- A gallery to display entries, duh
- Submission page for ... submissions
- Connection to some database with item info:
    - Title
    - URL
    - Screenshot
    - Tags/labels
